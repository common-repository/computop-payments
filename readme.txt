Use Computop Payments plugin for WooCommerce to provide an easy-to-install and use payment gateway integration for your payments.

== Description ==

• Computop Payments for WooCommerce payments by credit card, Amazon Pay, PayPal, EasyCredit and many more
• Easily accept digital payments in your online store with Computop Payments integration
• Computop Payments is a 3rd party service: https://www.computop.com / https://computop.com/uk/data-protection
• JavaScript from external domains can be added if specific payment methods are activated: www.computop-paygate.com , www.paypal.com , static-na.payments-amazon.com, static-eu.payments-amazon.com/checkout.js, static-eu.payments-amazon.com/checkout.js, static-fe.payments-amazon.com/checkout.js

== Available payment methods ==

• Amazon Pay
• Credit Card
• Direct Debit
• EasyCredit
• Giropay
• iDeal
• Klarna
• Paypal


== About Computop Payments ==

In order to use Computop Payments via this plugin, you'll need a contract with Computop.
Please contact sales via the form on https://www.computop.com/de/.
In case you are already a customer of Computop, we kindly ask you to get in touch with your contact person.
Following that, you'll receive your access data with which you'll be able to use the wooCommerce plugin.
You will then be able to to use Computop payments in your shop.
